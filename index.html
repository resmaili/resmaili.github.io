About

I am a senior scientist with expertise in remote sensing and meteorological applications. My work involves transitioning satellite observations into decision-support tools for forecasters and hazard managers.

To increase satellite data access and coding literacy amoung Earth scientists, I teach Python programming courses in my free time. As a companion to the courses, I published a python programming book through the American Geophysical Union (AGU).

Research
Precipitation, Air Quality, Satellite Soundings, Machine Learning, Cloud Computing
Book

Earth Observation using Python: A Practical Programming Guide.
The book is geared towards Earth scientists of all career levels who want to learn how to use Python for satellite data analysis. It covers the basics of Python programming and the visualization of complex satellite data using step-by-step examples. Some coding experience is helpful, but not required.

If you are interested in reading the book, you can:

  • Check if your library has eBook copies, access to the Wiley Library, or an AGU institutional subscription. If not, ask your library to purchase a copy.
  • Purchase a personal copy (AGU members get 35% off). It is also available in major online bookstores.
